# アセットバランス管理

投資ポートフォリオのアセットアロケーションを管理するシングルページWebアプリです。

## 機能

- **ファンド管理** — ファンドの追加・編集・削除・並び替え（ドラッグ＆ドロップ）
- **現在の配分確認** — ドーナツチャートで現在のアセット配分を可視化
- **目標アロケーション & 差分** — 目標比率との差分を計算し、リバランス金額を表示
  - 通常リバランス（売買あり）
  - 買い増しリバランス（追加投資額を指定して売却なしでリバランス）
- **取り崩しシミュレーション** — 金額または割合を指定して取り崩し後の配分をシミュレーション
- **データ管理** — JSON形式でのエクスポート・インポート・全データクリア

## 対応アセットタイプ

国内株 / 海外先進国株 / 新興国株 / 国内債券 / 海外先進国債券 / 新興国債券 / 海外REIT / 国内REIT / コモディティ

## 使い方

`index.html` をブラウザで開くだけで動作します。サーバーやビルド手順は不要です。

データはブラウザの `localStorage` に自動保存されます。

## 技術スタック

- HTML / CSS / JavaScript（バニラ、フレームワーク不使用）
- [Chart.js 4.x](https://www.chartjs.org/) — ドーナツチャート（CDN経由）
